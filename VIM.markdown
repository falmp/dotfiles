# vim cheat sheet

## folding

`zo` open fold
`zc` close fold
`zd` delete fold
`zf<Motion>` create fold (*i.e. zfa(* or *zfa5j*)
`zr` open 1 fold level (**r**educe)
`zm` close 1 fold level (**m**ore)
`zR` open all folds
`zM` close all folds
`<F9>` toggle fold
`<Space>` toggle fold (normal mode); create fold (visual mode)
